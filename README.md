# Music Store Data Analysis (SQL Project)
This repository contains a complete PostgreSQL database schema with a 50-row sample dataset and solutions to various business questions categorized into Easy, Moderate, and Advanced levels.
## Tech Stack Used
- Database: PostgreSQL
- Testing Platform: OneCompiler
## Repository Structure
- `Music_Store_schema_and_data.sql`: Script to drop, recreate, and populate all 11 tables.
- `Music_Store_queries_solutions.sql`: SQL codes for all 11 business analysis questions.
